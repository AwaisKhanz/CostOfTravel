Project OverviewCost of Travel is a deterministic travel intelligence platform consisting of 21 production-grade calculator tools that help travelers understand real financial outcomes of common travel scenarios — cancellations, refunds, baggage fees, delays, pet travel, insurance, and more.The platform is built on a strict policy-first, zero-hardcoding, server-time-controlled architecture. Every tool produces typed, versioned JSON output. No guessing. No AI in the logic engine. No assumptions.1.1 Core Principles (Never Violate These)Deterministic Only — same inputs always produce same outputPolicy-Normalized — all rules come from a normalized data layer, never hardcodedServer-Time Controlled — never trust client clock; server generates nowUTCVersioned Output — every tool output has schemaVersion: &apos;v1&apos;Namespaced Scenarios — all scenario IDs use dot notation (e.g., cancellation.free_24h)No AI in Logic Engine — LLM only explains output, never decides itFinancial Precision — all currency rounded to 2 decimals via round(value * 100) / 100Reconciliation Required — refundAmount + creditAmount + totalLoss = ticketPriceCannot Determine &gt; Wrong Answer — always return cannot_determine if policy is missing1.2 Technology StackLayerTechnologyPurposeFrontendNext.js App RouterServer Components, page routing, ISRReact Hook Form + ZodForm collection + validationTailwind CSSStyling (no MUI, no custom design systems)Logic EngineTypeScript onlyPure deterministic calculatorsData LayerDev #1 (Pre-built)Normalized airline/cruise/insurer policy storeAirport Dataourairports.com CSVIATA lookup, country/timezone mappingTrackingRybbitCustom events — typed, no free stringsLLM (optional)GPT-4o-mini (server-side)Explains tool output — never decidesCachingCloudflare + ISRPerformance at scale1.3 Airport Data SourceFor any tool requiring airport lookup (Tools #1, #6, #7, #8), download the static airport CSV from:https://ourairports.com/data/  →  airports.csvThis file is used to build the geography.ts module that maps: IATA code → country code → jurisdiction (US / EU / INTL) → timezone.
